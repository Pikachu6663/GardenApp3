Этот проект содержит библиотеку классов для работы с садом и цветами, а также консольное приложение для демонстрации их работы.

## Классы

- `Garden` - основной класс, представляет сад.
- `Flower` - дополнительный класс, представляет цветок в саду.

## Использование

1. Создайте объект `Garden`.
2. Добавьте несколько объектов `Flower` в сад.
3. Используйте метод `ShowAll()` для отображения всех цветов в саду.
